开发环境：ubuntu 11.10
开发语言：C++
编译器：g++
编译运行方法：
在linux操作系统下，进入代码目录，执行"make"命令，就会产生可执行文件main，运行即可

liushuo.h: 头文件，函数声明等
main.cpp: 主文件
PI.cpp: buffon针算法计算pi值
simplex.cpp: 单纯性算法的实现
RandomizedQuickSort.cpp: 随机快排算法
RandomizedSelect.cpp: 随机选择算法
RandomSampling.cpp: 随机取样算法
Utils.cpp: 各种常用函数定义
Queens.cpp: 求解八皇后问题的算法
PollardRho: Pollard rho 启发式算法
